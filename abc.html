UTF-8 Characters Many UTF-8 characters cannot be typed on a keyboard, but they can always be displayed using numbers (called entity numbers): A is 65 B is 66 C is 67 Example

I will display A B C

I will display A B C

Example Explained The element defines the character set. The characters A, B, and C, are displayed by the numbers 65, 66, and 67. To let the browser understand that you are displaying a character, you must start the entity number with &# and end it with ; (semicolon).
